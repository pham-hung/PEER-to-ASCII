# PEER-to-ASCII
PEER earthquake data (.AT2 ) to ASCII file (.txt)

This small program is used to convert PEER earthquake time history data (https://peer.berkeley.edu/peer-strong-ground-motion-databases) to ASCII file (.txt) file.
